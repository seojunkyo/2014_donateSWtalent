[SW 재능기부챌린지 공모전]

정보통신 산업진흥원에서 주관한 "SW 재능기부 챌린지"에 참가해 팀원과 어플리케이션, 웹페이지, 마케팅 활동을 통해 최우수상을 받아 

미래창조 과학부 장관상을 수상하였습니다.

저는 안드로이드 App개발과 FTP를 이용해 서버/클라이언트 사이의 파일전송 그리고 PHP를 통해 어플리케이션과 MySQL Database와 연동하는 역할을 맡았습니다.
(App <–> PHP <–> Database)

Database에서 응답으로 받는 형태는 자료간 읽기/쓰기와 기계의 분석에 용이한 JSON형태의 응답메세지로 보내 App 내부에서 JSON Object를 파싱해 데이터를 사용했습니다.

 또한 로그인 기능을 위해서는 Id와 Password를 HTTP POST 방식으로 전송하고 Database 내부의 비밀번호의 보안성을 높히기 위해 SQL 내부의 password 함수를 사용해 저장하였습니다.

그리고 LoginState를 저장하기 위해 안드로이드 내부의 자원을 사용할 수 있는 Shared Preference를 이용하여 LOGIN_STATE 변수를 통해 자동로그인 기능을 구현할 수 있었습니다.
